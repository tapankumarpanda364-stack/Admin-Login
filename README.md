## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/tapankumarpanda364-stack/ERP-Application.git
```

### Install Dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### Run the Project

```bash
# Frontend
npm run dev

# Backend
npm run dev
```
